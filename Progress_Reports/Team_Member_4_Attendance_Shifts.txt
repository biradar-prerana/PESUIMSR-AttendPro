2. MODULE DESCRIPTION

The Admin Attendance & Shifts module consists of the following parts:

2.1 Shift Management
    - Create new shifts with name and code
    - Set shift start and end times
    - Configure grace period (in minutes)
    - Set working days for each shift
    - Activate/deactivate shifts
    - Edit existing shifts
    - Delete shifts

2.2 Attendance Tracking
    - View all employee attendance records
    - Filter by date range
    - Filter by department
    - Filter by employee
    - Show check-in and check-out times
    - Calculate late arrivals based on shift timing
    - Calculate early departures

2.3 Attendance Management
    - Manual attendance entry
    - Edit existing attendance records
    - Delete attendance records
    - Mark absent employees manually
    - Add notes to attendance entries

2.4 Attendance Reports
    - Daily attendance summary
    - Monthly attendance reports
    - Department-wise attendance
    - Late arrival reports
    - Export to CSV format
    - Charts and visualizations

2.5 Attendance Simulator
    - Simulate check-in for testing
    - Simulate check-out for testing
    - Test different scenarios
    - View simulated results

3. TECHNICAL IMPLEMENTATION

3.1 Technologies Used
    - React.js for frontend interface
    - Node.js/Express.js for backend API
    - MongoDB for database
    - Chart.js for visualizations
    - React Router for navigation

3.2 API Endpoints Used
    - GET /api/shifts - List all shifts
    - POST /api/shifts - Create new shift
    - PUT /api/shifts/:id - Update shift
    - DELETE /api/shifts/:id - Delete shift
    - GET /api/attendance - Get all attendance
    - POST /api/attendance - Mark attendance
    - PUT /api/attendance/:id - Update attendance
    - DELETE /api/attendance/:id - Delete attendance

3.3 Key Files Created/Modified
    - src/pages/Shifts.js - Shift management page
    - src/pages/Attendance/Attendance.js - Attendance tracking page
    - src/pages/Attendance/AttendanceSimulator.js - Attendance simulator
    - Backend shift controller
    - Backend attendance controller
