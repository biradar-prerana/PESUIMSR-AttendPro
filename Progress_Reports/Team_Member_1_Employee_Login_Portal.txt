2. MODULE DESCRIPTION

The Employee Login and Employee Portal consists of the following parts:

2.1 Employee Login Page
    - A login form where employees enter their email and password
    - Validation of user credentials against the database
    - Display of error messages for invalid login
    - Remember me option for convenience
    - Forgot password link (UI only)
    - Demo credentials display for testing

2.2 Employee Portal (Dashboard)
    - Personal welcome message with employee name
    - Quick stats showing attendance summary
    - Today's check-in/check-out status
    - Recent attendance history
    - Quick action buttons

2.3 Employee Profile
    - View personal details (name, email, department, shift)
    - View employee ID and card number
    - View biometric enrollment status
    - Profile photo display

2.4 Employee Attendance View
    - View own attendance history
    - Filter by date range
    - View check-in and check-out times
    - View attendance status (present, absent, late)

2.5 Leave Management
    - Apply for leave (sick leave, casual leave, vacation)
    - Select leave dates
    - Enter reason for leave
    - View leave balance
    - View leave history
    - View status of leave applications (approved, pending, rejected)

2.6 Mark Attendance (Employee Self-Service)
    - Self check-in/check-out from portal
    - Real-time attendance marking
    - View current attendance status

3. TECHNICAL IMPLEMENTATION

3.1 Frontend Technologies Used
    - React.js for user interface
    - React Router for navigation
    - React Hot Toast for notifications
    - CSS for styling

3.2 API Endpoints Used
    - POST /api/auth/login - For user authentication
    - GET /api/employees/:id - Get employee details
    - GET /api/attendance/employee/:id - Get employee attendance
    - POST /api/attendance - Mark attendance
    - GET /api/leaves/employee/:id - Get employee leaves
    - POST /api/leaves - Apply for leave

3.3 Key Files Created/Modified
    - src/pages/Login.js - Login page component
    - src/pages/EmployeePortal/EmpDashboard.js - Employee dashboard
    - src/pages/EmployeePortal/EmpProfile.js - Profile view
    - src/pages/EmployeePortal/EmpAttendance.js - Attendance view
    - src/pages/EmployeePortal/EmpApplyLeave.js - Leave application
    - src/pages/EmployeePortal/EmpLeaveBalance.js - Leave balance
    - src/pages/EmployeePortal/EmpMyLeaves.js - Leave history
    - src/pages/EmployeePortal/EmpMarkAttendance.js - Self attendance
    - src/pages/EmployeePortal/EmpLayout.js - Portal layout
    - src/pages/EmployeePortal/EmpSidebar.js - Portal sidebar
